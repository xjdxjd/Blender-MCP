## 2. 代码优化方案

### 2.1 性能瓶颈识别方法

| 方法 | 工具 | 适用场景 | 产出 |
|------|------|---------|------|
| **cProfile 性能分析** | `cProfile` + `pstats` | 识别函数级热点 | 按耗时排序的函数调用统计 |
| **行级性能分析** | `line_profiler` | 精确定位耗时代码行 | 每行代码的执行时间和命中率 |
| **内存分析** | `memory_profiler` / `tracemalloc` | 识别内存泄漏和内存峰值 | 内存分配轨迹和增长趋势 |
| **WebSocket 延迟分析** | 自定义计时中间件 | 通信层延迟定位 | 请求-响应各阶段耗时分解 |
| **Blender 操作计时** | `bpy.app.handlers` + 计时装饰器 | bpy API 调用耗时 | 各 bpy 操作的执行时间 |

```python
import cProfile
import pstats
import io
from functools import wraps

def profile_tool(func):
    """
    MCP 工具性能分析装饰器。
    在 DEBUG 模式下自动记录工具调用的性能数据。
    """
    @wraps(func)
    def wrapper(*args, **kwargs):
        if not _is_debug_mode():
            return func(*args, **kwargs)

        pr = cProfile.Profile()
        pr.enable()

        try:
            result = func(*args, **kwargs)
        finally:
            pr.disable()

        s = io.StringIO()
        ps = pstats.Stats(pr, stream=s).sort_stats('cumulative')
        ps.print_stats(20)

        import logging
        logging.getLogger('blender-mcp.profile').info(
            f"工具 {func.__name__} 性能分析:\n{s.getvalue()}"
        )

        return result
    return wrapper

def _is_debug_mode() -> bool:
    from config.settings import LOG_LEVEL
    return LOG_LEVEL == 'DEBUG'
```

### 2.2 常见优化模式

#### 2.2.1 缓存模式

| 缓存场景 | 缓存策略 | 失效条件 | 预期收益 |
|---------|---------|---------|---------|
| 场景信息查询 | 内存缓存，TTL 1s | 场景变更 | 减少 80%+ bpy 查询 |
| 对象列表查询 | 内存缓存，TTL 1s | 对象增删改 | 减少 80%+ bpy 查询 |
| 网格哈希计算 | 对象级缓存 | 对象修改 | 大网格哈希计算从秒级降到毫秒级 |
| 材质属性查询 | 材质级缓存 | 材质修改 | 减少节点树遍历 |

#### 2.2.2 批处理模式

| 批处理场景 | 实现方式 | 预期收益 |
|-----------|---------|---------|
| 批量创建对象 | 单次 depsgraph 更新后批量创建 | 减少 50%+ 场景更新开销 |
| 批量变换操作 | 合并为矩阵运算后一次性应用 | 减少多次 bpy 调用 |
| 批量导出 | 单次场景遍历 + 循环导出 | 减少场景状态切换 |
| 批量布尔运算 | 排序后按依赖顺序执行 | 减少中间状态更新 |

```python
class BatchOperationManager:
    """
    批量操作管理器。
    将多个同类操作合并执行，减少场景更新次数。
    """

    def __init__(self):
        self._pending_creates: list[dict] = []
        self._pending_transforms: list[dict] = []

    def add_create(self, params: dict) -> None:
        self._pending_creates.append(params)

    def add_transform(self, params: dict) -> None:
        self._pending_transforms.append(params)

    def flush(self) -> dict:
        """
        执行所有待处理的批量操作。

        Returns:
            批量操作结果汇总
        """
        import bpy

        results = {'created': [], 'transformed': []}

        if self._pending_creates:
            with bpy.context.temp_override():
                for params in self._pending_creates:
                    result = _create_single_object(params)
                    results['created'].append(result)
            self._pending_creates.clear()

        if self._pending_transforms:
            for params in self._pending_transforms:
                result = _transform_single_object(params)
                results['transformed'].append(result)
            self._pending_transforms.clear()

        return results
```

#### 2.2.3 异步模式

| 异步场景 | 实现方式 | 预期收益 |
|---------|---------|---------|
| 渲染操作 | 异步渲染 + 进度回调 | 不阻塞 MCP 服务 |
| 大文件导入 | 异步导入 + 进度通知 | 不阻塞其他工具调用 |
| 状态同步 | 后台定时增量同步 | 不影响工具调用响应时间 |
| 模型检查 | 异步检查 + 结果缓存 | 长时间检查不阻塞 |

```
  -alpha.N    ：内部测试版
  -beta.N     ：公开测试版
  -rc.N       ：发布候选版

示例：
  0.1.0-alpha.1   第一个 alpha 版本
  0.1.0-beta.1    第一个 beta 版本
  0.1.0           第一个正式版本
  0.2.0           新增功能
  1.0.0           第一个稳定大版本
```

版本号决策规则：

| 变更类型 | 版本号变更 | 示例 |
|---------|-----------|------|
| 新增 MCP 工具 | 次版本号 +1 | 0.1.0 → 0.2.0 |
| 新增工具参数（向下兼容） | 次版本号 +1 | 0.2.0 → 0.3.0 |
| 修复 Bug | 修订号 +1 | 0.2.0 → 0.2.1 |
| 变更工具接口（不兼容） | 主版本号 +1 | 0.2.1 → 1.0.0 |
| 性能优化（无接口变更） | 修订号 +1 | 0.2.0 → 0.2.1 |

### 4.2 发布包构建脚本设计

```python
import os
import shutil
import subprocess
import hashlib
import json
from datetime import datetime

class ReleaseBuilder:
    """
    发布包构建器。
    自动化构建、打包、校验和生成流程。
    """

    def __init__(self, version: str, output_dir: str = "./dist"):
        self.version = version
        self.output_dir = output_dir
        self.artifacts: list[str] = []

    def build(self) -> dict:
        """
        执行完整的构建流程。

        Returns:
            构建结果摘要
        """
        os.makedirs(self.output_dir, exist_ok=True)

        self._run_tests()
        self._build_python_package()
        self._build_addon_package()
        self._generate_checksums()
        self._generate_release_notes()

        return {
            "version": self.version,
            "artifacts": self.artifacts,
            "build_time": datetime.now().isoformat(),
        }

    def _run_tests(self) -> None:
        """运行全量测试"""
        result = subprocess.run(
            ["pytest", "tests/", "-v", "--tb=short"],
            capture_output=True,
            text=True,
        )
        if result.returncode != 0:
            raise RuntimeError(f"测试未通过:\n{result.stdout}\n{result.stderr}")

    def _build_python_package(self) -> None:
        """构建 Python 包（wheel + sdist）"""
        subprocess.run(
            ["python", "-m", "build", "--outdir", self.output_dir],
            check=True,
        )
        for filename in os.listdir(self.output_dir):
            if filename.endswith(('.whl', '.tar.gz')):
                self.artifacts.append(os.path.join(self.output_dir, filename))

    def _build_addon_package(self) -> None:
        """构建 Blender 插件 ZIP 包"""
        addon_dir = "blender_plugin"
        zip_name = f"blender-mcp-addon-{self.version}.zip"
        zip_path = os.path.join(self.output_dir, zip_name)

        shutil.make_archive(
            base_name=os.path.join(self.output_dir, f"blender-mcp-addon-{self.version}"),
            format='zip',
            root_dir=addon_dir,
        )
        self.artifacts.append(zip_path)

    def _generate_checksums(self) -> None:
        """生成校验和文件"""
        checksums = []
        for artifact in self.artifacts:
            if os.path.exists(artifact):
                sha256 = hashlib.sha256(
                    open(artifact, 'rb').read()
                ).hexdigest()
                checksums.append(f"{sha256}  {os.path.basename(artifact)}")

        checksum_path = os.path.join(self.output_dir, "checksums.txt")
        with open(checksum_path, 'w') as f:
            f.write('\n'.join(checksums))
        self.artifacts.append(checksum_path)

    def _generate_release_notes(self) -> None:
        """从 CHANGELOG.md 提取当前版本的发布说明"""
        notes_path = os.path.join(self.output_dir, "RELEASE_NOTES.md")
        with open(notes_path, 'w') as f:
            f.write(f"# Blender-mcp v{self.version}\n\n")
            f.write(f"发布日期: {datetime.now().strftime('%Y-%m-%d')}\n\n")
            f.write("请参阅 CHANGELOG.md 获取详细变更记录。\n")
        self.artifacts.append(notes_path)
```
