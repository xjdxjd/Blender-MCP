## 安装步骤

### 方式一：开发安装

适用于开发者本地测试和调试。

#### 1. 克隆项目

```bash
git clone https://github.com/your-org/blender-mcp.git
cd blender-mcp
```

#### 2. 创建虚拟环境

```bash
# 使用 Blender 内置 Python
/path/to/blender/4.2/python/bin/python3.11 -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

#### 3. 安装依赖

```bash
pip install -r requirements.txt
pip install -r requirements-dev.txt
```

#### 4. 安装 Blender 插件

**Linux/macOS：**
```bash
mkdir -p ~/.config/blender/4.2/scripts/addons/
ln -s $(pwd)/blender_plugin ~/.config/blender/4.2/scripts/addons/blender_mcp
```

**Windows：**
```powershell
New-Item -ItemType Junction -Path "$env:APPDATA\Blender Foundation\Blender\4.2\scripts\addons\blender_mcp" -Target "$pwd\blender_plugin"
```

#### 5. 启用插件

1. 启动 Blender
2. 进入 `编辑` → `偏好设置` → `插件`
3. 搜索 "blender-mcp"
4. 勾选启用

### 方式二：生产环境安装

适用于生产环境部署。

#### 1. 下载发布版本

```bash
# 下载最新发布版
wget https://github.com/your-org/blender-mcp/releases/latest/download/blender-mcp-v1.0.0.tar.gz
tar -xzf blender-mcp-v1.0.0.tar.gz
cd blender-mcp-v1.0.0
```

#### 2. 系统级安装

```bash
# 安装到系统 Python
pip install -r requirements.txt

# 安装 Blender 插件到用户目录
mkdir -p ~/.config/blender/4.2/scripts/addons/
cp -r blender_plugin ~/.config/blender/4.2/scripts/addons/blender_mcp
```

#### 3. 配置 MCP 服务

创建配置文件 `~/.blender-mcp/config.yaml`：

```yaml
server:
  host: "127.0.0.1"
  port: 8765

blender:
  plugin_enabled: true
  auto_connect: true

logging:
  level: "INFO"
  file: "~/.blender-mcp/logs/server.log"

security:
  allowed_paths:
    - "/path/to/workspace"
    - "/tmp/blender-mcp"
```

## 配置说明

### 环境变量

| 变量 | 默认值 | 描述 |
|------|--------|------|
| `BLENDER_MCP_PORT` | 8765 | MCP 服务端口 |
| `BLENDER_MCP_HOST` | 127.0.0.1 | MCP 服务地址 |
| `BLENDER_MCP_LOG_LEVEL` | INFO | 日志级别 |
| `BLENDER_MCP_CONFIG` | ~/.blender-mcp/config.yaml | 配置文件路径 |

### 配置文件格式

```yaml
server:
  host: string           # 服务监听地址
  port: integer         # 服务监听端口
  max_connections: integer  # 最大连接数（默认 5）

blender:
  plugin_enabled: boolean   # 插件是否启用
  auto_connect: boolean     # 自动连接
  connection_timeout: integer  # 连接超时（秒）

logging:
  level: string          # DEBUG/INFO/WARNING/ERROR
  file: string          # 日志文件路径
  max_size: integer      # 单个日志文件最大大小（MB）
  backup_count: integer  # 保留的日志文件数量

security:
  allowed_paths:         # 允许的文件操作路径
    - string
  max_file_size: integer  # 最大文件操作大小（MB）
  enable_audit: boolean    # 启用审计日志

limits:
  max_objects: integer     # 最大对象数量
  max_operation_time: integer  # 最大操作时间（秒）
  rate_limit: integer      # 速率限制（操作/秒）
```

## 验证部署

### 1. 验证 Blender 插件

1. 启动 Blender
2. 检查控制台输出（窗口 → 切换系统控制台）
3. 应看到类似输出：
   ```
   Blender_mcp: Plugin enabled
   Blender_mcp: WebSocket server starting on port 8765
   ```

### 2. 验证 MCP 服务

```bash
# 测试 MCP 服务是否运行
python -c "import socket; s=socket.socket(); s.connect(('127.0.0.1', 8765)); print('MCP service is running')"
```

### 3. 验证 AI 助手连接

在 AI 助手中运行：
```
ping
```

应返回：
```json
{
  "success": true,
  "message": "pong",
  "version": "1.0.0"
}
```

### 4. 运行集成测试

```bash
pytest tests/integration/ -v
```

## 故障排查

### 常见问题

#### 问题 1：Blender 插件无法启用

**症状**：在 Blender 插件列表中找不到或启用时出错

**解决方案**：
```bash
# 检查插件目录
ls -la ~/.config/blender/4.2/scripts/addons/blender_mcp

# 检查 Python 错误
blender --python-expr "import addon_utils; addon_utils.module_bl_info('blender_mcp')"
```

#### 问题 2：WebSocket 连接失败

**症状**：AI 助手无法连接到 MCP 服务

**解决方案**：
1. 检查端口是否被占用：
   ```bash
   lsof -i :8765
   ```

2. 检查防火墙设置：
   ```bash
   # Linux
   sudo ufw allow 8765

   # macOS
   sudo firewallctl allow 8765
   ```

3. 检查 Blender 控制台是否有错误信息

#### 问题 3：操作超时

**症状**：操作执行时间过长或超时

**解决方案**：
1. 增加超时配置
2. 检查系统资源使用情况
3. 查看日志中的具体错误

### 日志位置

| 环境 | 日志路径 |
|------|----------|
| Linux | `~/.blender-mcp/logs/` |
| macOS | `~/.blender-mcp/logs/` |
| Windows | `%USERPROFILE%\.blender-mcp\logs\` |

### 调试模式

启用调试模式以获取更详细的信息：

```bash
export BLENDER_MCP_LOG_LEVEL=DEBUG
blender --python scripts/test_debug.py
```

## 生产环境配置

### 安全建议

1. **使用本地连接**：仅允许 localhost 连接
   ```yaml
   server:
     host: "127.0.0.1"
   ```

2. **配置路径白名单**：限制文件操作范围
   ```yaml
   security:
     allowed_paths:
       - "/path/to/workspace"
   ```

3. **启用审计日志**：记录所有操作
   ```yaml
   security:
     enable_audit: true
   ```

### 性能优化

1. **调整连接数**：根据用户数调整
   ```yaml
   server:
     max_connections: 10
   ```

2. **配置缓存**：减少重复计算
   ```yaml
   performance:
     enable_cache: true
     cache_ttl: 300
   ```

3. **监控资源使用**：使用 `htop` 或类似工具监控

### 备份策略

```bash
# 自动备份配置和日志
0 2 * * * tar -czf /backup/blender-mcp-$(date +\%Y\%m\%d).tar.gz ~/.blender-mcp/
```

### 升级指南

1. **停止服务**：
   ```bash
   # 停止 Blender
   ```

2. **备份数据**：
   ```bash
   cp -r ~/.blender-mcp ~/.blender-mcp.bak
   ```

3. **安装新版本**：
   ```bash
   pip install --upgrade blender-mcp
   ```

4. **验证升级**：
   ```bash
   blender_mcp --version
   ```
