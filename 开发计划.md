# Blender-mcp 客户端开发计划

## 阶段一：项目初始化与基础架构搭建

### 目标
建立项目基础结构，配置开发环境，实现最小可用的 MCP 服务与 Blender 插件连接。

### 任务清单
- [ ] 研究 Blender 4.2+ Python API 新特性与变更
- [ ] 创建项目目录结构
- [ ] 配置 Python 虚拟环境
- [ ] 编写 requirements.txt
- [ ] 实现基础 MCP 服务框架（server.py）
- [ ] 实现 Blender 插件基础结构（addon.py）
- [ ] 实现 MCP 服务与 Blender 插件的基础通信
- [ ] 编写第一个测试工具（ping/pong）

### 交付物
- 可运行的项目骨架
- 基础通信链路验证

### 预计时间
2-3 天

---

## 阶段二：3D打印建模核心功能开发

### 目标
实现3D打印建模所需的核心功能，包括对象创建、变换、修改器、模型修复、文件导入导出等。

### 任务清单
- [ ] 实现 create_object 工具
  - [ ] 支持创建基础网格（立方体、球体、圆柱体、平面、圆锥、圆环等）
  - [ ] 支持创建曲线和曲面
- [ ] 实现 transform_object 工具
  - [ ] 支持移动（location）
  - [ ] 支持旋转（rotation）
  - [ ] 支持缩放（scale）
- [ ] 实现 modify_mesh 工具
  - [ ] 支持布尔运算（union、difference、intersect）
  - [ ] 支持倒角（bevel）
  - [ ] 支持挤出（extrude）
  - [ ] 支持实体化（solidify）
- [ ] 实现 export_model 工具
  - [ ] 优先支持 STL 格式（3D打印标准格式）
  - [ ] 支持 OBJ 格式
  - [ ] 支持导出选项配置（缩放、坐标系等）
- [ ] 实现 import_model 工具
  - [ ] 支持 STL 格式
  - [ ] 支持 OBJ 格式
- [ ] 实现 check_model 工具
  - [ ] 检查非流形几何体
  - [ ] 检查法线方向
  - [ ] 检查壁厚
- [ ] 实现 repair_model 工具
  - [ ] 修复法线方向
  - [ ] 合并重叠顶点
  - [ ] 填充孔洞
- [ ] 实现 list_objects 工具
- [ ] 实现 get_scene_info 工具
- [ ] 编写单元测试

### 交付物
- 可通过 AI 助手创建和编辑3D打印模型
- 模型检查与修复功能
- STL/OBJ 导入导出功能
- 场景查询功能

### 预计时间
5-6 天
