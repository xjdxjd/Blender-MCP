# Blender-mcp 客户端开发计划

## 阶段一：项目初始化与基础架构搭建

### 目标
建立项目基础结构，配置开发环境，实现最小可用的 MCP 服务与 Blender 插件连接。

### 任务清单
- [ ] 研究 Blender 4.2+ Python API 新特性与变更
- [ ] 创建项目目录结构
- [ ] 配置 Python 虚拟环境
- [ ] 编写 requirements.txt
- [ ] 实现基础 MCP 服务框架（server.py）
- [ ] 实现 Blender 插件基础结构（addon.py）
- [ ] 实现 MCP 服务与 Blender 插件的基础通信
- [ ] 编写第一个测试工具（ping/pong）

### 交付物
- 可运行的项目骨架
- 基础通信链路验证

### 预计时间
2-3 天

---

## 阶段二：核心工具开发（对象操作）

### 目标
实现基础的 3D 对象创建与变换功能。

### 任务清单
- [ ] 实现 create_object 工具
  - [ ] 支持创建基础网格（立方体、球体、圆柱体、平面）
  - [ ] 支持创建灯光
  - [ ] 支持创建相机
- [ ] 实现 transform_object 工具
  - [ ] 支持移动（location）
  - [ ] 支持旋转（rotation）
  - [ ] 支持缩放（scale）
- [ ] 实现 list_objects 工具
- [ ] 实现 get_scene_info 工具
- [ ] 编写单元测试

### 交付物
- 可通过 AI 助手创建和变换基础 3D 对象
- 场景查询功能

### 预计时间
3-4 天

---

## 阶段三：高级功能开发（材质与渲染）

### 目标
实现材质设置、渲染控制等高级功能。

### 任务清单
- [ ] 实现 set_material 工具
  - [ ] 基础材质创建
  - [ ] 颜色、粗糙度、金属度设置
- [ ] 实现 render_scene 工具
  - [ ] 渲染参数配置
  - [ ] 触发渲染
  - [ ] 渲染结果保存
- [ ] 实现简单的修改器支持（subdivision surface）
- [ ] 完善错误处理和日志
- [ ] 编写集成测试

### 交付物
- 材质设置功能
- 渲染控制功能

### 预计时间
3-4 天
